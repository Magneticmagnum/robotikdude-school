Compiling:

To compile, navigate to the folder that contains the file GibbsRain.java. Run
the command:
   javac GibbsRain.java
This will compile the program GibbsRain.

Running:

To run the program GibbsRain, navigate to the folder that contains the file
GibbsRain.class. Run the command:
   java GibbsRain 100
This will run the program GibbsRain for 100 iterations. If you would like to
run more iterations, just change it too:
   java GibbsRain 10000

Information:

This program will compute the the Gibbs approximation of the Rain Network for
P(r|s,w).
